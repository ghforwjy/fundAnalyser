# 基金分析平台 - 界面设计方案

## 项目概述
本项目是一个公募基金数据分析与投资组合管理平台，提供基金数据查询、分析、投资组合管理等功能。

## 设计原则
1. **简洁高效**：界面清晰，操作流程简洁
2. **数据驱动**：以数据展示和分析为核心
3. **响应式设计**：支持不同屏幕尺寸
4. **深色主题**：采用深色主题，减少眼部疲劳

## 组件设计规范

### 颜色方案
- 主色调：青色 (#0ea5e9)
- 背景色：深色主题 (#0f172a)
- 边框色：半透明灰色
- 文本色：主文本、次要文本、禁用文本

### 间距规范
- 页面外边距：24px
- 卡片间距：16px
- 元素间距：8px, 12px, 16px

### 图标
- 使用 Lucide React 图标库
- 图标尺寸：16px, 20px, 24px

### 交互设计
- 按钮悬停：背景色变化
- 卡片悬停：轻微上浮、边框高亮
- 加载状态：旋转动画
- 成功/失败：Toast 提示

## 响应式设计
- 桌面端：侧边栏展开，多列布局
- 平板端：侧边栏可折叠，2-3列布局
- 移动端：侧边栏抽屉式，单列布局

## 问题分析

**根本原因**：Header.tsx 中的全局搜索框只是一个本地输入框，没有连接任何 API 或导航功能。用户输入搜索词后什么都不会发生。

## 修复方案

1. **修改 FundList.tsx** - 从 URL 参数读取搜索词（`?keyword=xxx`）
2. **修改 Header.tsx** - 搜索时跳转到 `/funds?keyword=xxx`
3. **修改 Layout.tsx 和 App.tsx** - 传递导航函数到 Header

## 修改文件

1. `app/src/pages/FundAnalysis/FundList.tsx` - 添加 URL 参数读取
2. `app/src/components/layout/Header.tsx` - 搜索跳转功能
3. `app/src/components/layout/Layout.tsx` - 传递 onNavigate
4. `app/src/App.tsx` - 传递 navigate 函数

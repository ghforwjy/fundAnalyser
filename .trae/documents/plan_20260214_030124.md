## 实施计划

### 第一步：创建fund_dividend表
```sql
CREATE TABLE IF NOT EXISTS fund_dividend (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    fund_code VARCHAR(10) NOT NULL,
    ann_date DATE,              -- 公告日期
    ex_date DATE,               -- 除息日
    pay_date DATE,              -- 派息日
    div_cash DECIMAL(10,4),     -- 每股派息
    record_date DATE,           -- 权益登记日
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(fund_code, ex_date)
);
```

### 第二步：创建同步器
1. **fund_portfolio_syncer.py**
   - 接口：fund_portfolio
   - 参数：start_date, end_date（支持日期范围）
   - 同步策略：按年度获取

2. **fund_div_syncer.py**
   - 接口：fund_div
   - 参数：ann_date（公告日期）
   - 同步策略：按年度获取

### 第三步：更新tables-status API
